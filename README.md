# 尚庭公寓

尚庭公寓是一个公寓租赁平台项目，包含移动端和后台管理系统

- 移动端：面向广大用户，提供找房、看房预约、租约管理等功能

- 后台管理系统：面向管理员，提供公寓（房源）管理、租赁管理、用户管理等功能。
